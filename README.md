# OrbitScan AI — Интеллектуальный сканер космических повреждений

*OrbitScan AI* — это инновационная система на базе искусственного интеллекта, предназначенная для автоматического обнаружения пробоин, царапин и других повреждений на обшивке космических аппаратов.

## Технологический стек
* *Нейросеть:* Модель классификации изображений, обученная с помощью TensorFlow.js.
* *Edge AI:* Анализ данных происходит локально на стороне пользователя, что обеспечивает высокую скорость и автономность.
* *Интеграция:* Система встроена в веб-интерфейс на платформе Tilda через кастомный HTML/JavaScript код.

## Как это работает
1. Пользователь загружает фотографию корпуса спутника.
2. Алгоритм Computer Vision анализирует изображение на наличие дефектов.
3. Система выдает мгновенный вердикт: "damage"(ОБНАРУЖЕНО ПОВРЕЖДЕНИЕ) или "upload"(КОРПУС ЦЕЛ).
